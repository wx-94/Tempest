phpMyAdmin -  Database Browser & Manager (for MySQL & MariaDB)

== Description ==

This plugin installs phpMyAdmin in Admin Dashboard. So, you no longer need to manually login PhpMyAdmin.

== Installation ==

= Method A =
1. Download & Extract the zip file.
2. Upload the container folder to "wp-content/plugins/"

= Method B = 
1. Enter your website Admin Dashboard
2. Mouseover "Plugins>Add New" and enter the plugin name
