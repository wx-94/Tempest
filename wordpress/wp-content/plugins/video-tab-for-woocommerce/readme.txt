== Description ==

Plugin to add the new tab for video or additional content like contact forms, shortcodes, important features and other useful information to WooCommerce products. This Video Tab For WooCommerce plugin extends WooCommerce to allow shop owners to add additional information on the tab of individual product pages at the right of the default tabs.

== Installation ==

1. Go to Plugins > Add New and search for Video Tab For WooCommerce, install and activate it.
2. We are good to go. No configuration needed.
3. Product Video tab is added to each product data information panel.


== Frequently Asked Questions ==

= Do this plugin works with all themes ? =

This plugin works with any theme that supports WooCommerce.

= Do plugin work with embed code only? =

You can either add embed code like iframe or use only url of the video directly from youtube or vimeo.

= Can I add multiple videos on the same tab ? =

Yes, you can add multiple videos on the same tab.

= Can I add contact form on the additional tab ? =

Yes, you can add shortcodes of contact forms on the additional tab added by plugin.

= Can I add HTML tags like h1, h2, span, etc ? =

Yes, you can add most of highly used HTML tags like h1-h6, span, p, iframe and more.
